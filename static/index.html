Bus Stop Endpoint

GET /busstop/<busstopid>
Parameters
Name Description
busstopid
string(6)
path 		Bus stop ID of specific bus stop.

Valid values:
"378204", "383050", "378202", "383049", "382998", "378237", "378233", "378230", "378229", "378228", "378227", "382995", "378224", "378226", "383010", "383009", "383006", "383004", "378234", "383003", "378222", "383048", "378203", "382999", "378225", "383014", "383013", "383011", "377906", "383018", "383015", "378207"
Responses Content type - application/json
Code - 200
Description - Successful operation.

Example Value:
Code - 404
Description - Invalid bus stop id.
